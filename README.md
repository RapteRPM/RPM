# RPM - Plataforma de Repuestos para Motocicletas

Plataforma web para facilitar la búsqueda de productos y servicios mecánicos en el mercado automotriz.

## Despliegue en Railway

Este proyecto está configurado para desplegarse automáticamente en Railway.

### Características

- Interfaz web moderna y responsiva
- Documentación por semestres del desarrollo del proyecto
- Enlaces a herramientas externas
- Acceso a formularios y bases de datos

### Tecnologías

- HTML5
- CSS3
- JavaScript
- Python (servidor HTTP simple)

## Desarrollo Local

Para ejecutar localmente:

```bash
python3 -m http.server 8000
```

Luego visita `http://localhost:8000`
